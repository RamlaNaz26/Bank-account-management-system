# Bank-account-management-system

## Description
This is a simple Bank Account Management System written in C++.
It allows users to create an account, deposit money, withdraw money,
and check account balance.

## Language Used
- C++

## Features
- Create bank account
- Deposit money
- Withdraw money
- Check balance

## How to Run
1. Open the project in any C++ compiler
2. Compile the file:
   g++ project1.cpp -o bank
3. Run:
   ./bank
